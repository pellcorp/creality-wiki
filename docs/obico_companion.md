# Obico Companion

I am playing loose with the naming, its called an Obico Agent and it is possible to install the Agent
on a RPI in the same network as your printer.

You must **not setup Obico** on a K1 series machine (which includes Ender 3 V3 KE, NebulaPad and Ender 5 Max) because
it will overwhelm the processor and cause serious stability issues.
