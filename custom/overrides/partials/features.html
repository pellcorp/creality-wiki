Why you should use SimpleAF

Open Source

Simple AF uses open source software and all its source code is open

Simple Configuration

All the macros and configuration are in the config directory and accessible to you, nothing is locked away in closed source blobs

Updated Klipper

The version of klipper we use although a fork, is regularly kept up to date with klipper3d and nothing we add changes the basic behaviour of klipper.

Vibrant Community

A vibrant discord community and approachable developer.
